ECE461 Group 17 module-registry

Get the packages from the registry:

POST /packages

Reset the registry:

DELETE /reset

Interact with the package with this ID:

GET /package/{id}

Update the content of the package:

PUT /package/{id}

Delete this version of the package:

DELETE /package/{id}

Upload the content of the package:

POST /package/{id}

Rate a package with id:

GET /package/{id}/rate

Create an access token:

/authenticate

Get any packages fitting the regular expression:

/package/byRegEx

Additional Features:

Return the history of this package (all versions)

GET /package/byName/{name}

Delete the history of this package (all versions)

DELETE /package/byName/{name}
